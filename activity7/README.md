# Robotics Management System

## Project Description
A Laravel-based web application designed to manage the educational infrastructure of a robotics academy. The system utilizes Class Table Inheritance to distinguish between Students, Teachers, and Administrators while handling the relationships between academic groups, courses, and the physical robotics kits assigned to those courses.
